# AI Powered Task Management System

A modern role-based Task Management System built with FastAPI, PostgreSQL, and React, featuring an AI Assistant powered by Google's Gemini.

## 🚀 Project Overview

This application allows:

- Admins to manage managers.
- Managers to manage employees and assign tasks.
- Employees to update task progress and log work hours.
- AI Assistant to provide role-specific assistance for Admins, Managers, and Employees.

## 🛠 Tech Stack

### Backend
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- JWT Authentication

### Frontend
- React
- Tailwind CSS

### AI
- Google Gemini API
